# Eficiência Digital — Portfolio

Landing page profissional do Pedro Henrique de Almeida.

## Stack
- HTML5 + CSS3 + JavaScript (vanilla), tudo inlined em `index.html`
- Google Fonts: Outfit + JetBrains Mono
- Formulário via Formspree (`mjgqzpnw`)
- Sem build step — abre direto no navegador

## Deploy
- **Netlify**: drag & drop a pasta, ou `netlify deploy --prod`
- **GitHub Pages**: Settings → Pages → Source: `main` / root
